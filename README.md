# HuggingFace-RAG-PDF-QA-System

📄 **Mini-RAG PDF Question Answering System** using **Hugging Face embeddings** and **Streamlit**.  

---

## Features
- Upload PDF and extract text automatically.  
- Split PDF into chunks and generate embeddings with **sentence-transformers**.  
- Store embeddings in **FAISS** for fast similarity search.  
- Ask questions and get answers generated by **Hugging Face LLM**.  
- Progress bar shows embedding generation.  
- Cached embeddings speed up repeated queries.

---

## Installation

```bash
git clone https://github.com/Preethi-MN/HuggingFace-RAG-PDF-QA-System.git
cd HuggingFace-RAG-PDF-QA-System
python -m venv myenv
# Windows
myenv\Scripts\activate
# macOS/Linux
source myenv/bin/activate
pip install -r requirements.txt
